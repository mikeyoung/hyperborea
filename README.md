# Spells of Hyperborea

Spells of Hyperborea is a tool for browsing an index of the hundreds of spells in the role playing game Hyperborea.  All code written for the application was distinct from all previous projects in CS50W.  The app is designed to be a part of a set of upcoming apps that will aid play in various ways such as character management, in-game weather conditions, etc.
